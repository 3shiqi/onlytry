# Release 0.1.0

Date: 2026-04-24

## Summary

First formal versioned release of `onlytry`.

## Included

- adaptive workout generation engine
- direct entry into the first exercise
- top planning tag filters for goal, duration, and fatigue
- refresh that always returns to exercise 1
- same-exercise set transition visibility
- time-based live countdown after pressing `CHECK`
- PWA and Cloudflare Pages deployment setup
- structured documentation and version management

## Known Constraint

- local production builds rely on the Rollup WASM fallback patch in `scripts/patch-rollup-native.cjs`
