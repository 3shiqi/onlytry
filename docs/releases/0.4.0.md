# Release 0.4.0

Date: 2026-04-24

## Summary

Introduced the first dual-mode app shell for training and external play logging.

## Included

- subtle `Train / Play` segmented control at the top of the app
- bottom global tab navigation for `Home` and `Calendar`
- `PlayLogger` screen with sport pills, duration slider, intensity slider, and TSS logging
- `CalendarView` screen for current TSS and rolling 7-day prescriptions
- extracted `WorkoutExecutor` so the training screen stays intact inside the new shell

## Behavioral Notes

- `TRAIN` mode keeps the existing workout executor as the Home content
- `PLAY` mode logs external sport sessions into `externalLogs` and `currentTSS`
- successful play logs show a lightweight toast and reset the form
