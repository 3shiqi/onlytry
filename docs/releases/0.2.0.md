# Release 0.2.0

Date: 2026-04-24

## Summary

Session execution now has a formal recovery mode between sets and between exercises.

## Included

- per-action `restIntra` and `restInter` metadata in the action library
- separate work timer and rest timer state in the executor
- circular bottom rest countdown with `Skip Rest`
- pale blue rest-state background and illustration accent shift
- refresh and top-tag changes clearing all live timers before restarting from exercise 1

## Behavioral Notes

- only time-based movements start a work countdown after pressing `CHECK`
- rep-based movements complete immediately, then enter rest
- finishing the final set of an exercise no longer slides instantly; it waits for `restInter`

## Known Constraint

- local production builds still rely on the Rollup WASM fallback patch in `scripts/patch-rollup-native.cjs`
