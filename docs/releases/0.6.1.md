# Release 0.6.1

Date: 2026-04-27

## Summary

Fixed stale PWA caching so deployed UI updates, including the monthly calendar, can replace older installed shells more reliably.

## Included

- versioned `service worker` shell and runtime caches
- network-first navigation updates for `index.html`
- immediate post-register update check in production boot

## Behavioral Notes

- users with an already installed home-screen app may still need one fresh reopen or refresh to pick up the new worker
- future deploys should no longer stay stuck on an older cached calendar shell under the same cache key
