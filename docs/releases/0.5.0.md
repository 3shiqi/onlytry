# Release 0.5.0

Date: 2026-04-24

## Summary

Rebuilt the Calendar tab into a system-load timeline that reacts to same-day play logging.

## Included

- new `CalendarPage` component with a hairline TSS progress bar
- 7-day date-based timeline from `Day 1` to `Day 7`
- prominent `Day 1` state for today's logged external sport
- automatic `Day 2` recovery slot after a same-day play log

## Behavioral Notes

- `calculateFluidCalendar(currentTSS)` remains the pure base periodization helper
- the logged-today override is applied in the page layer so other consumers can keep using the pure helper unchanged
